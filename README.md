# Domain classification

Domain classification is a Django app with machine learning engine to classify your domain into 3 categories: File-sharing, Communication and Other

How to use: 
```sh
$ cd apvera_app
$ python manage.py runserver
$ go to http://localhost:8000/urls/
```
### Development
The approach, I use tf-idf with ngram to vectorize URL and fit its into Logistic Regression to learn. 
Using threshold, I can category url into 3 topics
